# Quickstart: Validate Customers Mode UX Fixes

1. Open Inbox -> switch to **Customers** mode.
2. Select a customer with existing messages.
3. Scroll to bottom, then receive/send a new message:
   - timeline auto-scrolls to latest.
4. Scroll up (not near bottom), then receive/send a new message:
   - timeline does NOT jump.
5. Confirm unread UI:
   - list shows `Unread` badge only (no numeric count).
6. Open a customer row with unread:
   - unread badge clears after mark-as-read flow.
7. Switch to Conversations mode:
   - verify no behavioral changes in conversation mode UX.
